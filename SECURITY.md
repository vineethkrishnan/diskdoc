# Security Policy

## Supported Versions

Only the latest version of DiskDoctor is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of DiskDoctor seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, follow these steps:

1. Send an email to [vineethnkrishnan[at]gmail[dot]com](mailto:vineethnkrishnan@gmail.com).
2. Include a detailed description of the vulnerability and steps to reproduce.
3. We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.

Thank you for helping keep DiskDoctor safe!
